## Overview
This repository shows how to build, train, and evaluate a Transformer-based forecasting model using PyTorch and Streamlit. It focuses on:
- attention mechanisms, positional encoding
- preparing time-series for transformer input (sliding windows)
- encoder/decoder style forecasting
- training, evaluation, visualization and basic Streamlit UI

The dataset is downloaded automatically using `kagglehub`.
